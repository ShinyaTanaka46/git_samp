～情報システム科システムエンジニアコース～
イベント・科目紹介

  • Python

    • 近年飛躍的に注目されているプログラミング言語の一つ。
    「少ないコード量（文量）で簡単にプログラムがかける」「コードが読みやすい」というメリットから、
    本科では初めに習うプログラミング言語になります。

  • Java

    • 企業が大規模な業務システムを開発する際に、最も多く採用されているプログラミング言語です。
    JavaにはたくさんのAPIやライブラリが用意されています。
    「画像処理」「グラフィック」「構文解析」「データベース処理」等、
    目的に応じて効率的にシステム開発 を進めることができます！
    本授業は1年次で学習するJava基礎をもとに更なる技術向上を目指します。

  • SQL(データベース操作)

    • データベース言語の1つで、データベースの定義・操作を行います。
    SQLの種類は複数ありますが、本科では世界で最も広く普及されている「MySQL」を使用して学んでいきます。

  • 国家試験対策

    • 国家資格である基本情報技術者試験の午前試験免除認定校。
    入学後、自分のレベルに合わせて「ITパスポートクラス」「基本情報クラス」を選ぶことができ、
    経験豊富な講師陣の指導のもと、北東北No.1の合格率を誇る実力が評価されています。
    2019年度秋期では、岩手県内の専門学校生合格者３１名のうち、25名がモリジョビ生と高い合格率です！

  • イベント紹介

    • モリジョビには学校生活を思いっきり楽しんでたくさん想い出を作れるイベントが豊富！
    様々なイベントに参加することで、自然と自身の人間性を磨き上げることができます！
    充実した学生生活を送っていきましょう！
